Confocal Imaging for Retinal Surgery

IROS 2019

In this project, a novel semi-autonomous control framework is presented for enabling probe-based confocal laser endomicroscopy (pCLE) scan of the retinal tissue. With pCLE, retinal layers can be scanned and characterized in real-time for an improved diagnosis and surgical outcome prediction. This work proposes a auto-focusing algorithm that significantly improve the scanning outcome.

Surgical Scene Segmentation

Deep Learning Course Project

The goal of this project is to successfully segment 12 classes of objects in the surgical scene, using the dataset from EndoVis2018 Challenge. We proposed an architecture design named Super-Label which can improved the segmentation result by hierachical segmentation and a data splitting method named Mini-Data which encourages the diversity of data. We have demonstrated the efficacy of our approaches with a mean dice score of 0.78 on only 20% of the total data.

Vacuum Infusion End-Effector

Capstone Project

Current carbon fiber reinforced polymers (CFRP) manufacturing process is labour intensive, especially in the vacuum bagging stage. To improve the process, a proof of concept robot arm end-effector which can semi-automate the vacuum bagging process was proposed in collaboration with DLR and CARIS lab, UBC.
